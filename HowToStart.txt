
back :
se placer dans /backend puis :
$ npm init   (entry point server.js sinon enter)
$ npm install --save express
$ npm install --save mongoose
$ npm install --save body-parser
$ npm install -g nodemon
$ npm install --save mongoose-unique-validator
$ npm install --save jsonwebtoken
$ nodemon server


front :
se placer dans /Frontend puis :
$ npm install -D rollup rollup-plugin-terser @rollup/plugin-commonjs @rollup/plugin-node-resolve @rollup/plugin-babel @babel/preset-env
$ npm run build
$ npm run start

http://localhost:5000/
